# Print Perfect Calendar Creator

A web-based, React-powered application that imports ICS files, allows text editing, and mimics the look and feel of a physical desktop calendar.

## Setup Instructions

1. Make sure you have [Node.js](https://nodejs.org/) installed on your computer.
2. Open your terminal and navigate to this folder (`calendar-creator`).
3. Run `npm install` to download all the required dependencies (React, Vite, ical.js, xlsx).
4. Run `npm run dev` to start the local development server.
5. Open the URL provided in your terminal (usually `http://localhost:5173`) in your web browser.

## Building for Production
To create the final, deployable files for your web server, run:
`npm run build`
This will create a `dist` folder containing your final website files.
